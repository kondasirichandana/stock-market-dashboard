# Stock Market Analysis Dashboard 
 
Interactive financial analytics dashboard built with Python 
analysing 5 years of stock market data for AAPL, MSFT and TSLA 
 
## Tech Stack 
Python, pandas, yfinance, plotly, Streamlit 
 
## Key Findings 
- Tesla 1355 percent total return vs Apple 246 percent (2020-2024) 
- Tesla daily volatility 4.23 percent vs Apple 2.00 percent 
- Apple and Microsoft correlation 0.75 - move together as large-cap tech 
 
## How To Run 
pip install yfinance pandas plotly streamlit 
streamlit run dashboard.py
